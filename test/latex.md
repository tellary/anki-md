Formulas
--------

- Arithmetic progression

    $s_n = 1 + 2 + \ldots + n = \sum_{i = 1}^n i$
    
    $s_n = \frac {n(n + 1)}{2}$
    
    Recurrence:
    
    $s_n = n + s_{n - 1}$

- Geometric progression

    $s_n = \sum_{i=0}^n q^i$
    
    $s_n = {q^{n + 1} - 1 \over q - 1}$
    
    Recurrences:
    
    $s_n = s_{n - 1} + q^n$
    
    $s_n = 1 + q \cdot s_{n - 1}$
